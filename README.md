# リポジトリ概要
PythonでWebスクレイピングする

## Windowsでの実行方法

1.WindowsにPythonをインストール
* winget install python

1.実行に必要なライブラリをインストール
* pip install requests beautifulsoup4

1.実行方法
* python fetch_latest_tag_from_dockerhub.py
* python fetch_latest_tag_from_github.py
